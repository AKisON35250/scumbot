
SCUM Discord Status Bot (Render Ready)

Features
- Clean Discord Embed Design
- Online / Offline Indicator
- Player Count
- Ping
- Auto Update every 30 seconds
- Server Logo inside Embed
- Works with Render.com Web Service

SETUP

1. Upload this project to GitHub

2. Go to Render.com
Create a new Web Service

3. Settings

Build Command:
pip install -r requirements.txt

Start Command:
python bot.py

4. Edit config.json

Insert:
- Discord Bot Token
- Channel ID
- SCUM Server IP
- SCUM Server Port
- Logo URL

5. Deploy on Render

The bot will automatically update the embed every 30 seconds.
